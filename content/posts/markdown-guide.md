# 마크다운 가이드

이 블로그 엔진이 지원하는 문법을 정리했습니다.

## 제목

`#`부터 `######`까지 h1~h6 제목을 지원합니다.

### 소제목 예시

이렇게 세 번째 단계 제목도 사용할 수 있습니다.

## 목록

순서 없는 목록:

- 첫 번째 항목
- 두 번째 항목
- 세 번째 항목

순서 있는 목록:

1. 준비하기
2. 작성하기
3. 게시하기

## 인용

> 좋은 글은 다시 읽어도 좋은 글이다.

## 코드

인라인 코드는 `const x = 1`처럼 씁니다.

펜스 코드블록:

```
function greet(name) {
  return "Hello, " + name;
}
```
